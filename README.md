# interactive monologue web

chatbot-like interface for rendering localized chat bubbles based on user interaction

## assets

- `bubbles.{locale}.json` - required for rendering the _chat bubbles_
- `actions.{locale}.json` - required for defining the _actions_
- `metadata.json`
- `favicon.svg`

## web

only consists out of one page constituting a pfp and the chat itself

## i18n

- / -> renders with `en` as locale
- /at -> renders with `at` as locale
